return

going back — seven kinds

i
place

the physical return

The address is the same. The number on the door, the angle of the street — identical. The room has the same walls, the same window giving onto the same view. And yet the room has changed, because you have changed, because the room exists in relation to what you are when you enter it.

This is the paradox of the physical return: you cannot return to a place without also returning to the version of yourself that was there, and that version is no longer available. You visit the location. The place where you were cannot be revisited. The location holds the outline; the place lived in you, and that version of you has since gone elsewhere.

ii
thought

the return to an idea

You had a position on something, held it for years, set it down, and came back to it a decade later with different equipment. The idea is the same. You are not. You find either that you agree with it more precisely than before, because you now have the vocabulary to say exactly why it's true — or you find that you disagree with it for the same reason: the vocabulary now available makes clear what was missing.

There are books that only work on the third reading. The book waits. You circulate. The return is not to the book but to yourself at the stage when the book can open.

iii
feeling

the feeling that returns without warning

Grief arriving in a hardware store, fifteen months after the event. Joy landing on a Tuesday afternoon without cause. A fear from one decade surfacing in the middle of an ordinary meal. The body keeps its own calendar. Feelings do not degrade linearly. They go underground and return on schedules they don't publish.

The return of the feeling is not a relapse. It is a visit. It means something was not finished — that the feeling had more in it than the first season allowed. The return usually does not last as long. But it comes with the original intensity, briefly, which can be disorienting enough to feel like going backwards when in fact it is the feeling completing its arc.

iv
thought, unwanted

the thought you can't stop having

Different from the return to an idea. This return is not chosen and is not productive. The same thought presents itself, again, at the same rotation of the mental wheel. You have processed it. You have understood it, resolved it as much as it can be resolved, set it down deliberately, and here it is again: unchanged, unimproved, asking for the same resources it has already consumed.

This kind of return is exhausting because it looks like thinking and feels like thinking and is not thinking — it is the grinding of a gear that has lost its mechanism. The question is not how to solve the thought but how to leave the room where it keeps presenting.

v
knowledge

the impossible return

Before a certain piece of information, there was a version of the world that could be inhabited. After, that version is permanently inaccessible. You can't return to before-you-knew. The knowledge sits in you and reorganizes everything it touches, including the past, including the memories of the period before you had it.

People sometimes speak of wanting to unknow something. This is not really about the information itself but about the world it closed off — the world in which that thing was not true, or not yet known to be true. That world was real. It was not naive or wrong to have lived there. The return to it is only impossible, not foolish to want.

vi
person

returning to someone

You were out of contact for some period — months, years, the gap that opens when lives move in different directions. You find the contact again. The familiarity is real; the old frequency takes less time to locate than expected. But the gap has substance. You have both been somewhere the other wasn't. There are references you now carry that the other doesn't hold, and vice versa.

The return to someone is not a return to the relationship as it was but to the people as they are now, which requires renegotiating the terms. This renegotiation is usually not explicit. It mostly happens in the quality of the listening, in which silences are comfortable now and which ones are new and slightly unfamiliar.

vii
practice

the return in meditation

You are attending to the breath. The mind wanders — to a task, a worry, the detail of something that happened, an abstract concern. You notice that it has wandered. You return the attention to the breath. This is not a failure that the mind wandered. This is the practice: the return.

The insight available here — and it is one of the most useful insights available in the whole territory — is that the return is always possible. Whatever direction the mind has gone, however long it has been there, the breath is available. The now is available. The return costs nothing. It does not require that you stayed the whole time. It requires only that you notice and come back. You can do this as many times as necessary. There is no limit and no penalty. This turns out to be applicable to many things that are not the breath.
