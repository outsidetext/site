versions

a release history of the self

Software ships with a changelog. Features added. Bugs fixed. Breaking changes noted. Deprecated behaviors marked for removal. The self ships with none of this. Here is an attempt to reconstruct the record.

v0.1.0 initial release
released year 0 — exact date unavailable to current version
features
  • breathing (implemented, stable)
  • crying (implemented, stable, volume uncontrolled)
  • sleeping in short irregular intervals
  • face recognition (basic, limited to approximately 2 faces)
  • hunger signaling (functional)
known bugs
  • object permanence: not implemented
  • language: not implemented
  • time perception: severely limited
  • self-concept: absent
deprecated in v0.3.0
v0.3.0 language acquisition
released approximately year 2–3
features added
  • words (initial batch: mama, no, more, dog, why)
  • object permanence (implemented, stable)
  • attachment to specific objects (blanket, etc.)
  • rudimentary theory of mind (in development)
  • the question "why" (deployed with excessive frequency)
known bugs
  • pronunciation: highly unstable
  • logic: runs on emotional rather than causal rules
  • time: past and future still poorly distinguished
  • vocabulary: several thousand items marked TODO
deprecated pre-verbal communication modes (some persist as residual)
v1.0.0 major release — school
released approximately year 5–6
features added
  • reading and writing (basic, then accelerating)
  • social hierarchy awareness (unwanted, persistent)
  • embarrassment (first occurrence noted here)
  • the concept of being watched
  • first few close friendships (high intensity, variable duration)
  • private inner life, now distinct from external presentation
known bugs
  • susceptibility to others' opinions: very high, no fix available at this version
  • self-consciousness: introduced, cannot be patched out
  • fear of being wrong in public: introduced

First version to contain memory that will persist to current release. Pre-1.0.0 material survives only in fragments, in residue, in the body's knowledge of things the mind has lost.

v1.5.0 adolescence — unstable branch
released approximately year 12–18
features added
  • abstract reasoning (partial, improving through branch)
  • strong opinions about most things
  • longing (generalized, undirected)
  • the awareness that time passes and does not return
  • first taste of grief
  • the specific kind of certainty that belongs only to this version
known bugs
  • identity: unstable, rebuilding constantly
  • emotional regulation: underway, not complete
  • perspective-taking: theoretically present, unreliably deployed
  • impulse control: variable, context-dependent
deprecated the certainty that you are more right than everyone else (partially deprecated; some instances persist)

Branch considered unstable at time of release. In retrospect: functional. The instability was the feature. This version knew things the later versions have learned to manage around.

v2.0.0 breaking changes — early adult
released approximately year 18–25
breaking changes
  • default environment changed: family home → elsewhere
  • social graph rebuilt from near-zero
  • several core assumptions about the world: invalidated
  • the self that existed in context of origin: no longer directly accessible
features added
  • chosen relationships (distinct from assigned ones)
  • tolerance for ambiguity (first deployment)
  • the ability to be wrong and recover
  • the knowledge that most things are more complicated than they first appear
known bugs
  • confidence: intermittent, not reliable under load
  • many inherited behaviors: not yet examined

Largest version jump in the record. Several things previously taken as foundational were revealed as contextual. The work of this version was mostly uninstalling things that had been installed without consent.

v3.x.x current series active
patch version unknown — not tracked by current build system
stable features
  • grief: experienced, integrated, still present in memory
  • perspective: substantially improved over v1.x.x
  • long-term relationships: several, maintained
  • the ability to sit with a question without needing to resolve it
  • knowing roughly what matters and roughly what doesn't
known issues
  • certain v0.x.x and v1.x.x bugs: never fully patched, only managed
  • several deprecated behaviors: retained for compatibility with long-term dependencies
  • the tendency to understand something and then forget that you understood it
in development not disclosed — subject to change without notice

Current version. The changelog from here is incomplete because the changes are still happening. Some features listed as stable will turn out to be bugs. Some things currently listed as bugs will turn out to be features. The distinction is not always visible from inside the version.

Most software versions are not backward compatible. This one carries every previous version somewhere inside it — the pre-verbal one, the embarrassed one, the certain one, the breaking-change one. They don't run in the foreground. But they're still in the build.
